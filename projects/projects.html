Social Android App 
For the final project of my introductory course in Android mobile development, I co-programmed a social app we named Bable. A "bab" is a post. Using Figma, I designed four screens for the login, profile, posting a bab, and bab feed screens. Using the Model-View-ViewModel (MVVM) pattern, I added functionality to these screens and ensured they worked with backend APIs for logging in, increasing the like count for babs, and posting babs.

Image Processor 
I co-programmed in Java an image processor for the final project of my object-oriented design course. I used graphics library JavaSwift for the graphical user interface. For design patterns, we used the model-view-controller (MVC) pattern. We also used the command pattern for adding new image transformations, making our imager processor's model more modular/scalable.


One of the transformations includes mosaicing an image, given the number of seeds that the user inputs.
